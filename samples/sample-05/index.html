Recurring Report Generator

Week 4 M13 · Agentic AI Solo Agent
Claude.ai subscription required

Requires Claude.ai Pro or Team with Cowork access. Open Claude, start a Cowork session, and paste the agent prompt below. No API key or local Python setup needed.

What this does

This agent generates a complete structured report from raw data — daily, weekly, or on demand. It:

  • Accepts raw metrics (from a spreadsheet, pasted table, CSV, or description)
  • Computes derived KPIs: churn rate, ARPU, growth %, MoM trends
  • Builds a formatted markdown report with headline numbers, trend analysis, and commentary
  • Saves the report with a date-stamped filename
  • Surfaces the top 3 "so what" takeaways for leadership

Replaces 30–60 minutes of weekly analyst work. Paste your data, get your report.

Subscription required

Claude.ai Pro or Team subscription with Cowork access is required. Cowork lets Claude process structured data, compute metrics, generate formatted reports, and save files.

Claude.ai Pro Claude.ai Team Cowork feature
Step-by-step in Claude Cowork
Open claude.ai and sign in to your Pro or Team account.
Click "Cowork" in the left sidebar to start a new Cowork session.
Gather your raw data — paste a CSV, copy a spreadsheet table, or describe your numbers. The prompt below includes sample SaaS metrics you can replace with your own.
Paste this prompt into Cowork, replacing the sample data with your own metrics:
You are a recurring report generator agent. I'll give you raw metrics and you must build a complete report WITHOUT asking me questions. REPORT TYPE: Daily SaaS metrics REPORT DATE: [TODAY'S DATE] RAW METRICS (replace with your actual data): - New sign-ups today: 47 - Total active users: 1,284 - Churned users today: 3 - MRR this month: $24,350 - MRR last month: $22,100 - New paid conversions today: 8 - API calls today: 128,400 - API calls yesterday: 119,200 - Support tickets opened: 12 - Support tickets closed: 15 - Top feature used: Dashboard (38% of sessions) STEPS: 1. Compute these derived KPIs: - Daily churn rate = churned / total active × 100 - MoM MRR growth % = (this month - last month) / last month × 100 - ARPU = MRR / total active users - API call growth % vs yesterday - Support ticket resolution rate = closed / (opened + closed) × 100 2. Build a structured markdown report with these sections: ## Daily Report — [DATE] ### Headline Numbers (KPI cards: value + delta) ### Trend Analysis (2-3 paragraphs interpreting the numbers) ### Top 3 Takeaways (bullet points for leadership) ### Raw Data Reference (table of all inputs) 3. Flag any metric that is outside normal range: - Churn rate above 0.5% = WARNING - MRR growth below 5% MoM = WARNING - API growth negative = WARNING 4. Save the report as daily-report-[DATE].md Complete all steps without stopping.
Replace the sample metrics with your actual numbers. Replace [TODAY'S DATE] with today's date.
For recurring use, save this prompt as a template. Each day, paste the same prompt with updated numbers. The saved .md file is downloadable from Cowork.
