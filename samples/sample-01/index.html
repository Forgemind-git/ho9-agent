Research Brief Agent

Week 4 M13 · Agentic AI Solo Agent
Claude.ai subscription required

Requires Claude.ai Pro or Team with Cowork access. Open Claude, start a Cowork session, and paste the agent prompt below. No API key or local Python setup needed.

What this does

This agent automates the entire research brief workflow. You give it a topic, and it:

  • Plans 2–3 targeted search queries to cover the topic from multiple angles
  • Searches the web and identifies the most relevant sources
  • Reads the full content of the top sources
  • Synthesises findings into a structured markdown brief with executive summary, key findings, implications, and citations
  • Saves the finished brief to a file

A task that takes a human researcher 60–90 minutes completes in under 2 minutes.

Subscription required

Claude.ai Pro or Team subscription with Cowork access is required. Cowork lets Claude browse the web, run multi-step tasks, and save files — all the capabilities this agent needs.

No ANTHROPIC_API_KEY, no Python, no terminal. Just Claude.

Claude.ai Pro Claude.ai Team Cowork feature
Step-by-step in Claude Cowork
Open claude.ai and sign in to your Pro or Team account.
Click "Cowork" in the left sidebar to start a new Cowork session.
Choose your research topic (examples: "AI in healthcare", "remote work productivity", "sustainable packaging trends").
Paste this prompt into the Cowork input box and press Enter:
You are a research analyst running a multi-step agent job. I'll give you a topic and you must complete all steps WITHOUT asking me questions. Topic: [REPLACE WITH YOUR TOPIC] Steps to complete: 1. Plan 2-3 specific search queries that cover the topic from different angles 2. Search the web using each query and identify the 2 most relevant sources 3. Read the full content of those 2 sources 4. Synthesise what you found into a structured research brief in markdown with these sections: - Executive Summary (3-5 sentences) - Key Findings (bullet points with source citations) - Implications / So What - Sources (list all URLs consulted) 5. Save the brief as a file named [topic-slug]-brief.md Work through all 5 steps methodically. Show your reasoning at each step. Do not stop until the file is saved.
Replace [REPLACE WITH YOUR TOPIC] with your actual topic before pasting.
Watch Claude plan its queries, search, read sources, and write the brief. The file will be available to download when Cowork finishes.
