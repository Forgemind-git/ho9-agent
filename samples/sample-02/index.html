Lead Enrichment Agent

Week 4 M13 · Agentic AI Solo Agent
Claude.ai subscription required

Requires Claude.ai Pro or Team with Cowork access. Open Claude, start a Cowork session, and paste the agent prompt below. No API key or local Python setup needed.

What this does

This agent takes a plain list of company names and enriches each one with:

  • Industry / sector
  • Company size (headcount or revenue band)
  • Website URL
  • Key contact name and title
  • LinkedIn or other public profile URL

The result is a ready-to-use enriched CSV you can paste into your CRM. What used to take a sales rep hours of manual research is done in minutes.

Subscription required

Claude.ai Pro or Team subscription with Cowork access is required. Cowork gives Claude web-search capability so it can look up real company data for each entry in your list.

Claude.ai Pro Claude.ai Team Cowork feature
Step-by-step in Claude Cowork
Open claude.ai and sign in to your Pro or Team account.
Click "Cowork" in the left sidebar to start a new Cowork session.
Prepare your company list — a simple line-by-line list of names (e.g. Stripe, Notion, Linear, Vercel, Figma).
Paste this prompt into Cowork, replacing the company list with your own:
You are a lead enrichment agent. I'll give you a list of companies and you must research and enrich each one WITHOUT asking me questions. Companies to enrich: - Stripe - Notion - Linear - Vercel - Figma For EACH company, find and fill in: 1. Industry / sector 2. Company size (employee count or band: 1-50 / 50-200 / 200-1000 / 1000+) 3. Website URL 4. Founded year 5. Key contact: name and title (CEO or Head of Sales preferred) 6. Brief description (1 sentence) Steps: 1. Search the web for each company one by one 2. Extract the enrichment fields from search results 3. Present the results as a clean markdown table with columns: Company | Industry | Size | Website | Founded | Key Contact | Description 4. After the table, save it as a CSV file named enriched-leads.csv Do not skip any company. Work through the full list before presenting results.
Replace the example company list with your own names before pasting.
Claude will search the web for each company, fill in all fields, present a markdown table, and save the CSV for download.
