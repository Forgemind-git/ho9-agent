Agent That Does a Real Job

Week 4 M13 · Agentic AI Solo Agent
Done when
A multi-step job completed unattended.
What you build

You build an agent that does a real job — give it one trigger and it plans, uses tools, reads results, and completes a multi-step job unattended, with no human steering each step. Pick a chore you actually repeat and let the agent run the whole chain end-to-end. Ship the agent config/script + a recording of it running end-to-end + a README of the job it does. No API key needed — everything runs on your normal Claude.ai login (Pro or Team).

Pick a problem statement

Each sample is a real, hosted build — the agent config and a full unattended run all filled in. Open one live, or bring your own recurring job (get it approved first).

1Research Brief Agent

As an analyst you lose half a day researching every new topic by hand. Build an agent that, given a topic, gathers sources, reads them, and outputs a structured brief, running the full chain without you steering each step.

✓ one unattended run that takes a topic in and produces a sourced brief out Open live sample
2Lead Enrichment Agent

Your sales team imports company names but the rows are mostly empty. Build a lead-enrichment agent that takes a list of companies and fills in details (industry, size, website, a contact) from the web, step by step on its own.

✓ an unattended run that turns a bare list into an enriched one, recorded end to end Open live sample
3Page Change Monitor Agent

You manage pages or competitor sites and only notice changes when it is too late. Build a monitoring agent that checks a set of pages daily and reports what changed, deciding what is worth flagging without supervision.

✓ an unattended run that detects a change and produces a clear change report Open live sample
4Inbox Triage Agent

Your shared inbox piles up faster than you can sort it. Build an inbox-triage agent that reads incoming requests, categorises them, and drafts replies for your review, running the whole loop unattended.

✓ a batch that comes back sorted with draft replies ready for a human to approve Open live sample
5Recurring Report Generator

Leadership wants a recurring summary and assembling it by hand is a chore. Build a reporting agent that pulls data, builds a summary, and posts it to a channel, completing every step on its own.

✓ one unattended run from data pull to a finished report, no manual steps in between Open live sample
