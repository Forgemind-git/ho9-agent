How to use this repo

Click Use this template to create your own copy.

Pick the sample closest to your job, copy its agent-prompt.md brief into Cowork, and run it.

Adapt the brief to your own use case, then replace this README with what you built, the problem it solves, and how to run it.

Use your Claude.ai Pro plan wisely
  • Use the example prompt in each sample — it's already written and tested. Don't reinvent it.
  • One clear prompt beats lots of vague back-and-forth. Say what you want, with an example, in a single message.
  • Start a new chat when you switch tasks — long chats re-read every earlier message and use your limit faster.
  • Don't paste big files over and over. Paste once, then refer back to it.
  • If something works, keep it. Tweak it rather than regenerate from scratch.
